# Zonnestelsel

### Bouwen:
1. Open een terminal naar jouw keuze. Ik ben zeker dat dit werkt op Alacritty. Alleen macOS of Linux is echt ondersteund op dit moment. Mocht u Windows gebruiken, gebruik dan NIET cmd ivm Unicode die raar doet, maar bijvoorbeeld Alacritty.
2. `curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh` om Rust te installeren op macOS of Linux, mocht u Windows gebruiken, zie `https://doc.rust-lang.org/book/ch01-01-installation.html#installing-rustup-on-windows`. 
3. Download de code via `git clone https://github.com/sverrebesteman/zonnestelsel` of met de .zip direct van github.com
4. `cd` naar de folder
5. In de folder, typ `cargo run`. Voor de Windows liefhebber, sterkte met allemaal packages installeren. Ik ga ze hier niet linken. Live boot Linux of... Succes!
6. Laat het bouwen, en je bent binnen!
