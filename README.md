# Zonnestelsel

### Bouwen:
1. Open een terminal naar jouw keuze. Ik ben zeker dat dit werkt op Alacritty. (alleen macOS of Linux ondersteund op dit moment.)
2. Download de code via `git clone https://github.com/sverrebesteman/zonnestelsel` of met de .zip direct van github.com
3. `cd` naar de folder
4. In de folder, typ `cargo run`
5. Laat het bouwen, en je bent binnen!
