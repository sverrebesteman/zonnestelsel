# Zonnestelsel

### Bouwen:
1. Download de code via `git clone https://github.com/sverrebesteman/zonnestelsel` of met de .zip direct van github.com
2. Open een terminal naar jouw keuze, ik ben zeker dat dit werkt op Alacritty. (alleen macOS of Linux.)
3. `cd` naar de folder
4. In de folder, typ `cargo run`
5. Laat het bouwen, en je bent binnen!
